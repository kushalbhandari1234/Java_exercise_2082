Project name:Unit Converter – Convert between km/miles, kg/pounds, Celsius/Fahrenheit. Multiple Conversion Types – Distance (km ↔ miles), Weight (kg ↔ pounds), Temperature (C ↔ F). User Menu – Allows users to select the conversion type easily. Input Validation – Checks that user inputs are numeric and within reasonable ranges. Forward & Reverse Conversions – Supports both directions (e.g., km → miles and miles → km). Real-Time Conversion – Displays the converted value immediately after input. Repeat Option – Lets users perform multiple conversions without restarting. Exit Option – Allows users to quit safely. Clear Instructions & Feedback – Provides clear prompts and messages for better user experience.
